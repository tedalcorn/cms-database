# NYC CMS Provider Database

An interactive database of New York City's Crisis Management System (Cure Violence)
providers — city funding by year, IRS Form 990s, and staff contacts. A Vital City project.

**The page is password-protected** (client-side AES-GCM encryption). Enter the password to view.

Data sources: Checkbook NYC (payments), ProPublica / IRS Form 990s, and organizations'
public websites.
